#### 第一次提交

`f:`

`cd github\Blog`

`git init`   

`git checkout --orphan gh-pages`

`git add .`

`git commit -m "first post"`

`git remote add origin https://github.com/hopehook/Blog.git`

`git push origin gh-pages`



#### 后续提交

`git commit -m "提交备注"`

`git push -u origin gh-pages`
